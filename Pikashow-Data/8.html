How to Install Pikashow App on Firestick?

Complete steps to install Pikashow on Firestick.

  1. Now Go to Amazon App Store and Search “Downloader”
  2. Click the First “Downloader” App and Download and Install It
  3. After Installation, Open it and Give Permission to it asks.
  4. Now, Open the FireStick Settings. From the settings
  5. Then Click “My Fire TV”
  6. Then Choose “Developer Options”
  7. Then Click “Install Unknown Apps”
  8. Find Downloader and Click and Turn it “ON” (that allows you to install third-party apps)
  9. Then Open Downloader App from Apps Section in FireTV
  10. Then Click the search URL bar
  11. Type exact URL: https://bit.ly/3L8MTI9
  12. After this, install the file that you downloaded from the above-mentioned URL
